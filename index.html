Privacy Policy

Auto Scroller Chrome Extension — Effective date: June 25, 2026

✓ This extension collects zero personal data

Overview

Auto Scroller is a Chrome browser extension that enables users to automatically scroll web pages at a customizable speed using keyboard shortcuts. This policy explains what data the extension handles and how.

Data We Collect

Auto Scroller does not collect, transmit, store on any server, or share any personal data or user information whatsoever. Specifically:

  • No personally identifiable information (PII)
  • No browsing history or visited URLs
  • No page content or web activity
  • No authentication credentials
  • No location data
  • No device identifiers

Local Preferences Storage

The extension saves your preferences (scroll speed, keybind keys, overlay scale, and auto-popup setting) using chrome.storage.sync. This is Chrome's built-in extension storage, which:

  • Stores data locally in your Chrome profile
  • May sync across your own signed-in Chrome devices via your Google Account — this sync is managed entirely by Google and subject to Google's Privacy Policy
  • Is never accessible to websites you visit
  • Is never sent to any server we operate

The only data stored is: scroll speed (a number), start key name, pause key name, overlay scale (a number), and whether auto-popup is enabled (true/false). No URLs, page content, or personal details are ever stored.

Permissions Explained

The extension requests the following Chrome permissions:

  • activeTab — allows the extension icon to send a message to the currently active tab when clicked, to show or hide the overlay. No tab data is read or stored.
  • storage — allows saving your preferences (speed, keybinds, scale) using Chrome's built-in storage. See above.

The extension also injects a content script into all web pages (<all_urls>) so that the scrolling overlay and keyboard shortcuts function on any site you choose to use it on. This script only performs scrolling actions when you activate it — it does not read, record, or transmit any page content.

Third-Party Services

Auto Scroller does not integrate with, communicate with, or send data to any third-party services, analytics platforms, advertising networks, or data brokers.

Data Security

Because no personal data is collected or transmitted, there is no personal data at risk. Preferences stored via chrome.storage.sync are handled by Chrome's secure extension storage mechanism.

Children's Privacy

Auto Scroller does not collect any data from anyone, including children under the age of 13. The extension is a simple scrolling utility with no accounts, sign-ups, or data collection of any kind.

Changes to This Policy

If this privacy policy is ever updated, the new version will be posted at this URL with an updated effective date. Because the extension collects no data, material changes are unlikely.
